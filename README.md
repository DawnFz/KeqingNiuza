## 其他问题

### 如何加载祈愿数据

- 在游戏中打开任意祈愿历史记录
- 点击软件右上角加载数据

### 如何云备份

- 注册坚果云账户
- 账户信息->安全选项->第三方应用管理->添加应用
- 点击软件右上角的云，填写账户和**应用密码**
- 坚果云账户和用户密码会加密保存在本地
