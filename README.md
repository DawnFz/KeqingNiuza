## 其他

### 如何加载祈愿数据

- 在游戏中打开任意祈愿历史记录
- 点击软件右上角加载数据
- 短时间添加多个Uid：个人信息->更换Uid->添加Uid->加载数据

### 如何云备份

- 注册坚果云账户
- 授权第三方应用，[授权教程](https://help.jianguoyun.com/?p=2064)
- 点击软件右上角的云，填写账户和**应用密码**
- 坚果云账户和用户密码会加密保存在本地

### 如何导入6个月之前的数据

关于->导入数据
